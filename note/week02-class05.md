# 1교시

```java
a. int[] arr[];
b. int[] arr = {1,2,3,}; // 마지막의 쉼표‘,’는 있어도 상관없음.
c. int[] arr = new int[5];
d. int[] arr = new int[5]{1,2,3,4,5}; // 두 번째 대괄호[]에 숫자 넣으면 안됨.
e. int arr[5]; // 배열을 선언할 때는 배열의 크기를 지정할 수 없음.
f. int[] arr[] = new int[3][];
```

**d, e는 안되는 문법!**
